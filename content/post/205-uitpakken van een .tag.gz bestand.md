Uitpakken van een *.tar.gz bestand gaat in linux als volgt:
```bash
tar xvzf bestand
```

Parameter | Uitleg
--------- | ------------------
tar       | archiving utility
xvzf      | x = extract
          | v = verbosely list files processed
          | z = filter the archive through gzip
          | f = use file
